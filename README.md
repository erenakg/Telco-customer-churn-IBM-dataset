# Telco Customer Churn Analysis

Bu proje, telekomünikasyon sektöründeki müşteri kaybını (churn) analiz etmeyi amaçlamaktadır. Projede IBM Telco Customer Churn veri seti kullanılarak veri analizi ve makine öğrenmesi modelleme çalışmaları yapılmıştır.

## Proje İçeriği ve Dosyalar

- **churn.ipynb**: Ana analiz ve modelleme çalışmalarının yer aldığı Jupyter not defteri.
- **Old Models/**: Projenin önceki aşamalarına ait çeşitli analiz ve model denemelerini barındıran klasör (`P2P_Proje_Analizi.ipynb`, `telco_customers_churn2.ipynb` vb.).
- **churn_env/**: Projenin çalıştırılması için gerekli kütüphaneleri içeren Python sanal ortamı (virtual environment).

## Başlangıç

Projeyi kendi bilgisayarınızda çalıştırmak için sanal ortamı aktif edebilir ve Jupyter Notebook üzerinden `.ipynb` dosyalarını inceleyebilirsiniz.
